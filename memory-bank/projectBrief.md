# Project Brief: Homey Stelpro STZW402+

## Goal
Create a Homey app to support the Stelpro STZW402+ (Z-Wave) baseboard heater.

## Repository
https://github.com/Simon-CR/homey-Stelpro-STZW402-plus

## Scope
- Initialize a new Homey app.
- Implement Z-Wave driver for Stelpro STZW402+.
- Support core features:
  - Thermostat mode (Heat/Eco/Off)
  - Target temperature
  - Current temperature
  - Power consumption (if supported)
  - Z-Wave parameters configuration.

## Success Criteria
- App installs on Homey.
- Device can be paired.
- Temperature and modes can be controlled.
