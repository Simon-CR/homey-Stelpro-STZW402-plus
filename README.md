# Homey Stelpro App

This app adds support for Stelpro Z-Wave devices to Homey.

## Supported Devices

- **Stelpro Ki Thermostat (STZW402+)**
  - Z-Wave Plus Baseboard Heater Thermostat
  - Supports:
    - Current Temperature
    - Target Temperature
    - Thermostat Mode (Heat, Eco)

## Installation

1. Install the app from the Homey App Store (when available).
2. Or install via CLI: `homey app install`

## Development

- Clone the repository.
- Run `npm install`.
- Run `homey app run` to test.
